Archivos incluidos:
- index.html
- css/styles.css
- js/app.js
- assets/  (vacío - puedes añadir imágenes si quieres)

Cómo publicar en GitHub Pages (rápido):
1. Crea un repositorio público en GitHub (ya lo hiciste).
2. En tu repositorio, haz click en "Add file" -> "Upload files".
3. Sube todos los archivos del paquete (el contenido del ZIP), es decir: index.html, la carpeta css, la carpeta js y assets.
4. Commit changes.
5. Ve a Settings -> Pages -> Source -> Branch: main, Folder: / (root). Save.
6. Espera 1-2 minutos y abre: https://TU_USUARIO.github.io/NOMBRE_REPOSITORIO/

Nota: la app usa localStorage para guardar el historial y las preferencias. Si quieres que te arme el ZIP con una imagen/logo específico, súbelo y te lo incluyo en assets/.
